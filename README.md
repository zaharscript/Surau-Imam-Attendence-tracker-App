# 🕌 Surau Imam Attendance Tracker App

A modern web application for managing and tracking imam attendance for daily prayers at a surau or mosque.

Built using React, Firebase, and Tailwind CSS.

</div>

---

# ✨ Features

- 📅 Daily prayer attendance tracking
- 👥 Multi-user attendance management
- 🔐 Firebase Authentication
- ☁️ Real-time Firestore Database
- 📱 Responsive mobile-friendly UI
- ⚡ Fast and lightweight interface
- 📊 Attendance history records
- 🌐 Online deployment with Vercel

# 📂 Run Locally

## Prerequisites

- Node.js installed

---

## 1. Clone Repository

```bash
git clone https://github.com/zaharscript/Surau-Imam-Attendence-tracker-App.git
```

---

## 2. Navigate Into Project

```bash
cd Surau-Imam-Attendence-tracker-App
```

---

## 3. Install Dependencies

```bash
npm install
```

---

## 4. Setup Environment Variables

Create a `.env` file in the root folder and add:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```

---

## 5. Run Development Server

```bash
npm run dev
```
